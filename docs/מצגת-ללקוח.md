## שקופית 3 — האחסון / "השרת" (Vercel)

**מה זה Vercel?**
חברה שמאחסנת את האתר ומגישה אותו לגולשים. במקום שרת פיזי שצריך לתחזק,
Vercel עושה הכל אוטומטית: מאובטח, מהיר, ועם גיבוי.

**מה זה אומר בפועל?**

- אין שרת לתחזק, אין עדכוני אבטחה ידניים, אין "נפל השרת" באמצע הלילה
- האתר נטען מהר בכל מקום בעולם (רשת שרתים גלובלית)
- כל עדכון תוכן עולה לאוויר תוך שניות
- **עלות התחלתית: חינם** (מספיק בהחלט לאתר תדמית)

## שקופית 6 — דומיין (כתובת האתר)

**מה זה דומיין?** הכתובת שאנשים מקלידים, למשל `tzahi-tzedaka.co.il`.

- עלות: בערך ‎$15–20 לשנה (תלוי בסיומת .com / .co.il וכו').

### דרך ג' — המפתח מטפל בהכל 🤝

**חינם מול בתשלום:**

- להתחלה ולתחזוקה — **התוכנית החינמית מספיקה** לאתר כזה.
- אם רוצים לצרף כמה אנשים לצוות ניהול או יכולות מתקדמות — יש תוכנית Pro
  (בערך ‎$20 לחודש). **לא נדרש** לאתר תדמית רגיל.

> השורה התחתונה: צחי פותח חשבון, מקבל בעלות, וזה שלו לתמיד — בלי עלות חודשית.

# נספח טכני (למפתח בלבד — לא לשקופיות)

## הגדרת המייל (Resend)
ב-Vercel → Project → **Settings → Environment Variables** מגדירים:

| משתנה | חובה? | ערך |
|---|---|---|
| `RESEND_API_KEY` | חובה | המפתח מ-resend.com (מתחיל ב-`re_`) |
| `RESEND_TO` | אופציונלי | יעד המייל. ברירת מחדל: `site.email` (`Zedaka1@gmail.com`) |
| `RESEND_FROM` | אופציונלי | כתובת השולח. ברירת מחדל: `onboarding@resend.dev` |

- בלי `RESEND_API_KEY` הטופס מחזיר `missing_config` ונופל אוטומטית לוואטסאפ/טלפון.
- עד אימות דומיין ב-Resend: שולחים מ-`onboarding@resend.dev`,
  ו-`RESEND_TO` חייב להיות המייל שאיתו נפתח חשבון ה-Resend.
- אחרי אימות דומיין: מגדירים `RESEND_FROM` לכתובת בדומיין (למשל `office@tzahi-tzedaka.co.il`).
- הקוד: [src/app/actions.ts](../src/app/actions.ts)

## חיבור דומיין
- דרך א' (קנייה ב-Vercel): Project → **Domains** → חיפוש וקנייה → מתחבר אוטומטית.
- דרך ב' (ספק חיצוני): Project → Domains → Add → להזין את הדומיין →
  להגדיר אצל הספק את ה-DNS לפי ההוראות (A record `76.76.21.21` או Nameservers של Vercel).

## העברת בעלות / שיתוף
- שיתוף: Project → Settings → **Members** (דורש Team/Pro לכמה חברים).
- העברה מלאה: Project → Settings → **Transfer** → לחשבון/טים של הלקוח.
